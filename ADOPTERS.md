# Who's using TESAIoT — ใครใช้ TESAIoT บ้าง

If TESAIoT is helping you or your organization, we would love to list you here.
This is **an invitation, never an obligation** — your rights under Apache-2.0 do
not depend on it. Being listed is simply a way to share visibility with the
community: you get a little spotlight, and others get the confidence of seeing
real adopters. Listing implies no endorsement in either direction, and you can
remove yourself any time.

## How to add yourself — วิธีเพิ่มชื่อ

Either is fine:

1. **Open a pull request** adding one row to the table above with your
   organization name, a short note on how you use TESAIoT, and an optional link.
2. **Open a GitHub Discussion / issue**, or contact us via
   <https://www.tesaiot.com>, and we will add you.

Please add only an organization you are authorized to represent.

> ทำได้ทั้งสองทาง: (1) **เปิด pull request** เพิ่มหนึ่งแถวในตารางด้านบน พร้อมชื่อองค์กร คำอธิบายสั้นๆ
> ว่าใช้ TESAIoT ทำอะไร และลิงก์ (ถ้ามี) หรือ (2) **เปิด GitHub Discussion / issue** หรือติดต่อเรา
> ผ่าน <https://www.tesaiot.com> แล้วเราจะเพิ่มให้ — กรุณาเพิ่มเฉพาะองค์กรที่คุณมีสิทธิ์เป็นตัวแทน
